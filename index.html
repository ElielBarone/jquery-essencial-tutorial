Jquery

O jquery é muito poderoso quando se trata de manter telas dinâmicas em html via javascrit, e por muito tempo foi absoluto, até a chegada do Angular, mas nem mesmo o angular matou o jquery, pois o angular mantém uma versão mini do jquery em seu código, e quando você adiciona a dependência do jquery, o Angular passa a utilizá-la, em fim, se você quer ter um bom domínio em angular, estude Jquery antes

A variável $ representa o jquery, e você deve utilizar os seletores do jquery para realizar as seguintes tarefas à partir da ferramenta de desenvolvedor do chrome, na aba console:

tag selector

1 - Seleione todos os inputs

$('input');

tag selector

2 - Seleione todos os labels

$('label');

class selector

3 - Para selecionar um objeto através de sua classe é muito simples, pois nada muda em relação aos seletores que vimos até agora, basta que você troque o valor entre parênteses por '.nome-da-classe'. Então selecione todos os elementos que possuem a classe 'text-danger'

$('.text-danger');

class selector

4 - Você sabia que o comando .removeClass('class-name') remove uma classe css, assim como o comando .addClass('text-info') adiciona uma classe a um componente específico. Troque a cor dos labels que estão em vermelho para azul, você pode utilizar a classe 'text-info' para obter este resultado

$('.text-danger').removeClass('text-danger').addClass('text-info');

attribute selector

5 - Selecione apenas o input do tipo numérico

$('[type=number]');

attribute selector filtered string

6 - Selecione todos os elementos da sessão pessoa, ou seja: 'pessoa-nome' e 'pessoa.age'

$('[sessao|=pessoa]');

append

7 - Uma das funções mais valiosas supridas pelo jQuery é a 'append', como é de se imaginar, ela serve para que possamos adicionar trechos de código em nosso site, por exemplo:
'$('div').append('<h1>Hello Word</h1>');
O nosso formulário tem uma div que servirá para exibirmos mensagens ao usuário, ela possui a classe 'message-container'. Adicione a seguinte mensagem a este container: Preencha os campos e clique em salvar!

$('.message-container').append('<div class="alert alert-info">Preencha os campos e clique em salvar!</div>');

empty

8 - Para remover o conteúdo de uma div, você pode usar o método 'empty'. Este método muitas vezes pode ser confundido com o método 'remove', mas há uma diferença crucial entre eles, enquanto o primeiro remove o conteúdo do elemento selecionado, o segundo remove o próprio elemento juntamente com seu conteúdo. Remova agora a mensagem que você acabou de adicionar.

$('.message-container').empty();

hide

9 - Talvez você não precise remover e adicionar um certo elemento, você precisa que ele apenas fique invisível, para isso você pode utilizar a função 'hide'. Apesar de não ser uma boa prática esconder botões, seria melhor deixá-los desabilitados, esconda o botão salvar utilizando a função 'hide'.

$('.btn-primary').hide();

show

10 - Para mostrá-lo novamente você deve usar a função 'show'. Você sabia que é possível adicionar um argumento que torna essa ação animada? experimente mostrar o botão que escondeu, aplicando o argumento '1000', que é a quantidade de milisegundos. Lembre-se que existem outras alternativas para esta ação, como a função 'fadeIn'.

$('.btn-primary').show(1000);

val

11 - Para formulários dinâmicos, é muito importante que consigamos escrever e ler valores de um determinado imput, e é muito simples ler o valor de um input, basta que você utilize a função val(), e para escrever val('hello world'). Escreva agoora o seu nome no input 'nome'.

$('#name').val('Seu Nome');
